# 🚀 Machine Learning & Deep Learning Algorithms Implementation

A comprehensive repository showcasing the implementation of core **Machine Learning** and **Deep Learning** algorithms using real-world datasets. This project demonstrates strong fundamentals in supervised learning, model evaluation, and neural network architectures.

---

## 📌 Overview

This repository contains structured implementations of essential ML and DL algorithms, ranging from basic regression models to advanced deep learning architectures like CNNs and RNNs.

It is designed to:

* Build a strong foundation in machine learning concepts
* Understand deep learning architectures
* Apply models on real-world datasets (IMDB sentiment analysis)
* Evaluate model performance using standard metrics

---

## 🧠 Algorithms Covered

### 🔹 Machine Learning

* Linear Regression
* Multiple Linear Regression
* Logistic Regression
* k-Nearest Neighbors (k-NN)
* Support Vector Machine (SVM)
* Regularization Techniques:

  * Lasso
  * Ridge
  * Elastic Net

---

### 🔹 Deep Learning

* Artificial Neural Networks (ANN)
* Convolutional Neural Networks (CNN)
* Recurrent Neural Networks (RNN)
* LSTM & GRU

---

### 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
